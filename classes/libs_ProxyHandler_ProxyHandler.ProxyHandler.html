Class ProxyHandler<T>

Private Readonly_proxyMap

_proxyMap: WeakMap<object, unknown> = ...

A map to store proxies of objects to reuse them instead of unnecessarily creating new proxies for the same objects.

Remarks

  • The key is the object (proxy target) and the value is the proxy itself.
  • A WeakMap is used to prevent memory leaks => weak references to objects ensure that they can be garbage collected when they are no longer needed or referenced elsewhere.
  • Defined in src/libs/ProxyHandler/ProxyHandler.ts:24

_updateKeyValue: ((key: string, value: unknown) => void)

A delegate function for updating key-value pairs.

Remarks

  • If the proxy is used to update key-value pairs on the original object, this delegate function is used to update the key-value pairs on a different change tracking object.

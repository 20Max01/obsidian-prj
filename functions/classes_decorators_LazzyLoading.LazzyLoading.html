Function LazzyLoading

  • LazzyLoading<ClassInstType, PropertieType, InputType>(get, set?): ((target: ClassInstType, propertyKey: string | symbol) => void)

  • A decorator to lazily load a property in a class. The property is initialized using the provided get delegate when accessed for the first time. Optionally, a set delegate can be provided to customize the setting behavior of the property.

    Type Parameters

    • ClassInstType

      The type of the class instance.

    • PropertieType

      The type of the property to be lazily loaded.

    • InputType

      The type of the input value used in the setter delegate.

    Parameters

    • get: GetDelegate<ClassInstType, PropertieType>

      A delegate function to retrieve the property value.

    • Optionalset: SetDelegate<ClassInstType, InputType, PropertieType> | "Readonly"

      An optional delegate function or Readonly. If Readonly, no setter is defined. If undefined, a simple setter is used. If a function, the function is used as the setter.

    Returns ((target: ClassInstType, propertyKey: string | symbol) => void)

    A decorator function to be applied on the class property.

      • (target, propertyKey): void

      • Parameters

        • target: ClassInstType
        • propertyKey: string | symbol

        Returns void

    Remarks

    This decorator does not handle asynchronous loading or error handling. Asynchronous operations and errors should be managed within the provided delegates.

    Example

    class Example {
      \@LazzyLoading(
        (context) => {
          console.log("Loading value...");
          return 42;
        },
        (value, context) => {
          console.log("Setting value to", value);
          return value;
        }
      )
      public myProperty!: number;
    }

    const example = new Example();

    // The value is loaded only when myProperty is accessed for the first time
    console.log(example.myProperty); // Output: "Loading value..." 42

    // The setter is called when myProperty is set
    example.myProperty = 100; // Output: "Setting value to 100"
    console.log(example.myProperty); // Output: 100

cache

  • get cache(): default[]

  • Get the metadata cache

    Returns default[]

    Description

    This method returns the metadata cache as an array of FileMetadata objects. The FileMetadata object contains the file and the cached metadata.

    Remarks

    • The reference of the array is returned and will be the same on every call.
    • If the cache is updated, the array is emptied and repopulated with the current values from the metadata cache.
    • You can use the array permanently, but you should not rely on the order of the entries.
    • Defined in src/interfaces/IMetadataCache.ts:14
